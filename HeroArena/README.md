# Hero Arena

## Prérequis
- Visual Studio 2022+
- .NET 8
- SQL Server avec SSMS

## Packages NuGet
- Microsoft.EntityFrameworkCore.SqlServer 8.0.0
- Microsoft.EntityFrameworkCore.Tools 8.0.0

## Installation
1. Cloner le repo
2. Ouvrir HeroArena.sln
3. Lancer l'application
4. Cliquer Paramètres BDD
5. Entrer : Server=localhost\SQLEXPRESS;Database=ExerciceHero;Trusted_Connection=True;TrustServerCertificate=True;
6. Cliquer Tester puis Initialiser les données
7. Se connecter avec admin / admin123

## Identifiants par défaut
- Username : admin
- Password : admin123
